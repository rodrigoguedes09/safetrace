Getting Started

1. Create an Account

First, create an account to get access to the API. 

curl -X POST "https://your-domain.com/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "full_name": "Your Name",
    "password": "YourSecurePassword123"
  }'

2. Get an API Key

Use your credentials to create an API key. 

curl -X POST "https://your-domain.com/auth/bootstrap?email=your@email.com&password=YourPassword123" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My API Key",
    "description": "Production use"
  }'

3. Analyze a Transaction

Use your API key to trace a transaction's risk profile. 

curl -X POST "https://your-domain.com/api/v1/compliance/trace" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: sk_your_api_key_here" \
  -d '{
    "tx_hash": "0x7c9e8f2a3b4c5d6e7f8a9b0c1d2e3f4a5b6c7d8e9f0a1b2c3d4e5f6a7b8c9d0e",
    "chain": "ethereum",
    "depth": 3
  }'

API Reference

POST /api/v1/compliance/trace

Analyze a blockchain transaction for risk indicators.

Request Body

Parameter Type Required Description
tx_hash string Yes Transaction hash to analyze
chain string No Blockchain network (default: ethereum)
depth integer No Trace depth 1-10 (default: 3)
